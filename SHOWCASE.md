## How to add yours

1. Have you been running matins on a real project for at least a week? Add yourself.
2. Open a PR with:
   - Your project name
   - Domain profile(s) it uses (see [docs/DOMAINS.md](docs/DOMAINS.md))
   - Live URL or repo (if public)
   - One-line note (optional — what worked, what you'd change, anything)
3. Optional but highly valuable: drop an anonymized `MATINS.md` snapshot into `examples/` as part of the same PR.

We don't gate this — if your project runs matins, you can list it. We may move highly-active or unusual entries to a separate "Featured" section over time.
